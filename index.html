Scientific Data Analysis

The software tools for analysis X-ray data are collected and distributed in a sister project, scikit-xray, which has its own separate documention. An important feature of scikit-xray is its generic design, completely uncoupled for the particulars of data collection at NSLS-II. It can be used just as easily on data collected at any facility.
