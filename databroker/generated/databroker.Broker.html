class databroker.Broker(mds, reg=None, plugins=None, filters=None, auto_register=True)[source]¶

Unified interface to data sources

Eventually this API will change to __init__(self, hs, **event_sources)

Parameters:

mds : object

implementing the ‘metadatastore interface’

reg : object

implementing the ‘assets interface’

auto_register : boolean, optional

By default, automatically register built-in asset handlers (classes that handle I/O for externally stored data). Set this to False to do all registration manually.
