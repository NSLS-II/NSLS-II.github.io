bluesky.spec_api.a3scan¶

bluesky.spec_api.a3scan(*args, time=None, md=None)¶

Scan over one multi-motor trajectory.

Parameters:
  • *args –
    patterned like (motor1, start1, stop1, ...,
    motorN, startN, stopN, intervals)

    where ‘intervals’ in the number of strides (number of points - 1) Motors can be any ‘setable’ object (motor, temp controller, etc.)

  • time (float, optional) – applied to any detectors that have a count_time setting
  • md (dict, optional) – metadata
