dataportal.broker.get_images¶

dataportal.broker.get_images(headers, name)¶

Load images from a detector for given Header(s).

Parameters:
  • headers (Header or list of Headers) –
  • name (string) – field name (data key) of a detector

Example

>>> header = DataBroker[-1]
>>> images = Images(header, 'my_detector_lightfield')
>>> for image in images:
        # do something
