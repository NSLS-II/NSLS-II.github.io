Installation¶

PyXRF is supported for Python 3.6 and 3.7 in Linux/Mac/Windows systems. The easiest way to install PyXRF is to load it into a Conda environment from nsls2forge Anaconda channel. Installation instructions are identical for all supported OS.

  1. Install Conda. Select Python 3.7 version. Anaconda package is recommended to casual users because it installs the entire scientific Python stack. Advanced users may consider installing Miniconda since it is a much smaller download (~400 MB vs. ~20 MB).

    If Conda is already installed on your computer, proceed to the next step.

  2. Open a terminal (Linux/Mac) or start Anaconda Prompt from Windows Start Menu (Windows).

  3. If you are using existing Conda installation, update Conda:

    conda update -n base -c defaults conda

    If you are installing PyXRF on a Windows platform, close and restart Anaconda Prompt.

    Note

    The following instructions work in an environment with Conda 4.6 or newer. If PyXRF is installed in an older Conda environment then different command syntax must be used on some platforms (particularly Windows). Refer to the Conda documentation for more details.

  1. Install PyXRF from conda-forge or nsls2forge Anaconda channels.

    PyXRF is currently available from the conda-forge Anaconda channel. Create new Conda environment:

    conda create -n pyxrf-env python=3.7 -c conda-forge

    Activate the new environment:

    conda activate pyxrf-env

    Install PyXRF:

    conda install pyxrf -c conda-forge

    PyXRF package is also maintained at the nsls2forge Anaconda channel. Installation from nsls2forge involves similar steps:

    conda create -n pyxrf-env python=3.7
conda activate pyxrf-env
conda install pyxrf -c nsls2forge

    It is not recommended to mix packages from conda-forge channel and nsls2forge/defaults channels in the same environment, since they may be incompatible.

Starting PyXRF¶

  1. Open a terminal (Linux/Mac) or start Anaconda Prompt from Windows Start Menu (Windows).

  2. Activate Conda environment that contains PyXRF installation (in our example pyxrf-env):

    conda activate pyxrf-env

  3. Start PyXRF by typing:

    pyxrf

Updating PyXRF¶

  1. Open a terminal (Linux/Mac) or start Anaconda Prompt from Windows Start Menu (Windows)

  2. Activate Conda environment that contains PyXRF installation (in our example pyxrf-env):

    conda activate pyxrf-env

  1. Update PyXRF from the same Anaconda channel (conda-forge or nsls2forge) that was used for the original PyXRF installation:

    conda update pyxrf scikit-beam -c conda-forge

    or

    conda update pyxrf scikit-beam -c nsls2forge

Deactivating Conda environment¶

conda deactivate
